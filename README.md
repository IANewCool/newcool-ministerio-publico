# NewCool Ministerio Publico - Fiscalias Chile

Portal educativo sobre el Ministerio Publico y fiscalias en Chile.

## Demo

[https://ministerio-publico.newcool.io](https://ministerio-publico.newcool.io)

## Caracteristicas

- Buscador de fiscalias
- Guia de como denunciar delitos
- Derechos de las victimas
- Seguimiento de causas
- Informacion de fiscales

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
